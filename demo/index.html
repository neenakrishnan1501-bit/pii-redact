PII Redact Demo

Test the TypeScript library purely in your browser using DefaultMatchers.

Note on Default Matchers: The library uses strict structural matching by default to avoid false positives (like redacting generic math equations or order IDs). Test these formats:
  • Phone: 10-digit formats (e.g., 123-456-7890 or (555) 123-4567)
  • SSN: Exactly 9 digits formatted as 3-2-4 (e.g., 123-45-6789)
  • Credit Card: 16-19 valid digits (e.g., 4111222233334444)
  • Email: Standard email (e.g., admin@example.com)
  • Date of Birth: Standard date format (e.g., 1990-01-01 or 01-01-1990)
  • IPv4 / Mentions: e.g., 192.168.1.1 or @username
  • NLP Entities: Normal sentences with Names, Organizations, or Locations (e.g., John lives in Chicago and works for Microsoft.)
